# UI 配色分类与命名

基于你收集的 4 张 UI 视觉参考图，整体可归纳为“柔和浅色 + 粉彩分类 + 深色重点面板 + 小面积高亮”的视觉体系。适用于后续制作网页、App、后台仪表盘、课程平台、健康数据面板和卡片式功能页面。

## 配色总览

| 分类名称 | 风格定位 | 适用场景 | 主要色彩气质 |
|---|---|---|---|
| 柔光新拟物 | 柔软、低对比、有触感 | 天气、智能家居、轻工具 | 暖白、浅灰、蜂蜜黄 |
| 健康暖动感 | 温暖、活力、数据友好 | 健身、健康、运动看板 | 米灰、石墨黑、柠檬黄、珊瑚红 |
| 教育粉彩卡片 | 年轻、轻松、模块清晰 | 在线教育、课程平台、内容分类 | 薄荷绿、樱花粉、奶油橙、薰衣草紫 |
| 学习深色对比 | 专业、聚焦、强调进度 | 学习后台、任务系统、课程统计 | 深靛蓝、荧光黄、柔紫、浅米黄 |

## 板块建议

| 板块 | 视觉处理 | 推荐颜色 |
|---|---|---|
| 顶部导航 | 留白、低边界感、轻图标 | Cloud White、Ink Black |
| 侧边栏 | 图标化、圆形选中态 | Ivory Panel、Graphite Night、Honey Glow |
| 核心总览卡 | 大圆角、深浅对比强 | Deep Indigo、Cloud White、Neon Lime |
| 数据图表区 | 卡片化、柔和渐变 | Ivory Panel、Warm Halo、Pale Violet |
| 分类课程卡 | 粉彩分区 | Blush Course、Cream Apricot、Lavender Study、Aqua Skill |
| 进度与状态 | 小面积高亮 | Active Lemon、Coral Burn、Neon Lime |
| 搜索与筛选 | 胶囊形、浅底、弱阴影 | Cloud White、Pale Taupe |

## 使用规则

1. 每个页面最多选择 1 个背景色、1 个主强调色、2 到 4 个分类色。
2. 强调色只用于按钮、进度条、当前状态、徽标和关键数据点。
3. 深色面板只放核心内容，不要和粉彩卡片大面积混用。
4. 粉彩色适合做模块分类，不建议用作正文文字。
5. 强调色面积建议控制在页面的 5% 到 10%。
6. 新拟物风格需要搭配阴影、圆角和高光。
