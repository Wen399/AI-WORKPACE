# Skill：柔和低饱和新拟物网页 UI 设计

## 适用场景
适用于后台系统、数据看板、管理平台、分析仪表盘、SaaS 产品页等网页 UI 设计。

## 设计风格
采用柔和、低饱和度的新拟物风格。整体视觉应轻盈、干净、温和，避免高对比、高饱和、强烈撞色。

## 配色规范
- 主背景使用浅灰、米白、暖白等低饱和色。
- 强调色使用浅橙、暖黄、淡金色等柔和暖色。
- 辅助色可使用浅灰、灰蓝、雾面白、淡米色。
- 所有颜色需保持协调统一，不要使用突兀的高饱和颜色。
- 文本颜色使用深灰或中性灰，避免纯黑。

## 布局规范
- 左侧为统一导航区域，所有导航、菜单、筛选器、切片器均放置在左侧。
- 左侧导航支持展开与收起，可使用图标按钮控制。
- 页面顶部放置核心卡片指标区域。
- 卡片指标需固定停留在上方，并整体居中对齐。
- 所有卡片组件之间保持一致间距。
- 页面主体区域用于放置图表、列表、数据表格和可视化组件。

## 卡片与组件规范
- 卡片使用大圆角矩形。
- 卡片背景接近页面背景，但通过柔和阴影区分层级。
- 卡片、指标、图表组件右下方增加轻柔投影，营造立体悬浮感。
- 可适当添加左上高光与右下阴影，形成新拟物质感。
- 阴影不能过重，应保持柔和、自然、轻盈。
- 图表区域保持简洁，避免复杂装饰。

## 数据可视化规范
- 图表颜色以浅橙、浅黄、低饱和灰色为主。
- 折线图、柱状图、环形图、热力图等均需保持柔和视觉。
- 图表背景使用浅色卡片。
- 关键数据可使用浅橙色强调。
- 避免使用过多颜色分类，保持整体统一。

## 交互与细节
- 按钮、筛选器、标签、输入框采用圆角设计。
- 按钮可使用内阴影或外阴影表现按压感。
- 悬停状态可以增强阴影或轻微提高亮度。
- 收起导航时仅显示图标，展开时显示图标与文字。
- 当前选中导航项使用浅橙色高亮。

## 输出要求
生成网页 UI 时，需满足：
1. 低饱和浅灰与浅橙为核心配色。
2. 左侧导航与切片器可展开、可收起。
3. 顶部卡片指标固定并居中排列。
4. 卡片组件间距统一。
5. 卡片和图表右下方具备柔和阴影。
6. 整体风格干净、极简、柔和、具有新拟物立体感。
