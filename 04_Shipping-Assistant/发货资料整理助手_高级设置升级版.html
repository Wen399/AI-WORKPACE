发货资料整理助手

上传发货清单和资料压缩包，直接输出为本地文件夹，避免再次打 ZIP 导致压缩包无效。
本地浏览器处理 · 直接写入文件夹
1. 读取发货清单识别 出库单号 / 工厂 / 物流 / 三方仓外箱贴，并支持按 Sheet 设置站点和物流资料文件名。
2. 扫描压缩包识别 OBxxxx_仓库 文件夹、内部 ASNxxxx_OBxxxx.pdf，以及表格填写的货代箱唛/入仓单文件。
3. 自动回填将 PDF 前缀 ASN 编号写入对应 OB 行，并按表格文件名复制货代箱唛/入仓单。
4. 输出文件夹直接生成 发货文件 / 工厂 / 发货明细 + 站点 / 物流 / OB文件夹，公共物流资料会复制到对应OB。

发货清单 Excel

支持多工作表读取：可按每个工作表设置站点，例如 Sheet1=US、Sheet2=UK。留空则使用表内站点列。

工作表站点设置
每个 Sheet 最多对应一个站点。工具会优先使用这里填写的站点覆盖表内站点，留空则使用表内站点列；如果表内也没有站点，则不启用站点层级。

发货资料压缩包 ZIP

压缩包内包含 OB单号+仓库 的文件夹，外箱贴 PDF 在对应 OB 文件夹内部。

高级设置
请使用新版 Chrome / Edge；执行时会要求选择一个输出文件夹。
输出结构：发货文件_整理结果_时间 / 工厂 / 发货明细_工厂.xlsx + 站点 / 物流 / OB文件夹。站点可从表内站点列读取，也可按 Sheet 手动设置；没有站点时自动降级为：工厂 / 物流 / OB文件夹。不会生成空站点或空物流文件夹。若发货清单命名列填写“箱唛文件名 / 入仓单文件名”，工具会从同一个 ZIP 中查找并复制到对应 OB 文件夹；可在高级设置中控制是否删除辅助列、是否平铺物流资料、是否保留原始装箱清单。

执行日志

这版不再生成二次 ZIP，而是直接写入你选择的输出文件夹。匹配规则：从 OB 文件夹名和 PDF 文件名中提取 OB+数字；外箱贴使用 PDF 文件名第一个下划线前的内容，例如 ASN202606230000006_OB2606234576.pdf 会回填 ASN202606230000006。多工作表可在上传区设置站点，工具会优先按 Sheet 站点拆分。发货清单可增加“货代箱唛”“入仓单”两列，同一个文件名可填写在多个OB行，工具会自动复制到多个OB文件夹；导出的工厂发货明细是否删除这两列，可在高级设置中控制。物流资料文件名采用“OB上下文安全匹配”：优先完整文件名匹配，兼容下载(1)/(2)、副本/copy、空格/连接符/中英文符号差异、.xlx/.xls/.xlsx 误写；如果表格只填了部分文件名，只会在当前OB范围内做前后缀匹配，不再做全局包含模糊匹配。辅助资料列会严格识别“箱唛文件名 / 入仓单文件名”，不会再把“物流=海运/卡航”当作文件名。
